/*-----------------------------------------

  ポーカーサンプルプログラム

------------------------------------------*/

大学でC言語の実習が始まった初期に作ったCのサンプルプログラムと思われます。
まだアセンブラ脳が抜けてない頃に作ったコンソール上での簡単なポーカーのサンプルです。
役判定に2進数を使っているあたりが特にそれを感じますね。


※面倒なのでCPUのカード交換と、役が同じだった場合は引き分けにしてるようです。
※コンソールの設定によっては画面がバグって表示されます。

・皆さんなら役判定方法、どのように書きますか？
・忌憚のない意見をどうぞ。

========================================
cygwinなら以下の方法でコンパイルできます(gccが入ってることが前提)

1.githubからpoker.cで保存

2.gcc poker.c -o poker.exe

g++ poker.c -o poker.exe <-- g++だとエラーがでます

コンパイラのバージョンによってはgccでもエラーでるかもしれません。
